Getting started with Neon

Neon administration web application

Neon comes installed with Jetty, and will run on the port you s[ecified for the httpdPort parameter in the config, by default 8083. You can access the main page at http://localhost:8083/neon/hostlist.jsp Note: It looks best when viewed in Firefox or Safari

Running Neon

Neon contains two scripts - runneon and runneonconsole
runneon is normal script started from the command line
runneonconsole opens up a frame where Neon's output is redirected to.

Going further

More information can be found in the docs directory or at the Neon project site (http://neon.dev.java.net)
